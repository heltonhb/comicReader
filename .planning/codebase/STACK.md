# Stack

## Core
- **Frontend Framework:** React 18
- **Language:** JavaScript (ES Modules, JSX)
- **Styling:** Tailwind CSS v3
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Libraries
- **Routing:** react-router-dom
- **State Management:** zustand
- **Comic Reading & Viewing:** react-pageflip, react-zoom-pan-pinch
- **Virtualization:** react-virtuoso

## Backend / Services
- **Backend-as-a-Service:** Firebase 12
- **Analytics:** Mixpanel

## Tooling
- **Build Tool:** Vite 7
- **Linting:** ESLint 9
- **PWA:** vite-plugin-pwa
- **PDF Conversion Script:** pdf2pic (via Node.js script)
