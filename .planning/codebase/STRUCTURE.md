# Structure

## Diretórios Principais

- `/.planning/`: Arquivos centrais do framework de planejamento e agentes.
- `/public/`: Assets estáticos não processados pelo Vite.
- `/scripts/`: Scripts Node.js para ferramentas de backoffice (ex: `convert-pdf-to-webp.js`).
- `/source_hqs/`: Diretório provável que armazena os PDFs brutos ou originais antes da conversão.
- `/src/assets/`: Recursos de imagens, SVGs e CSS extra.
- `/src/components/`: Componentes visuais UI reaproveitáveis.
- `/src/hooks/`: Hooks customizados de React.
- `/src/store/`: Configuração de stores globais utilizando Zustand.
- `firebase.json` & `.firebaserc`: Definições de deploy e emuladores do Firebase.
