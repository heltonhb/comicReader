# Firebase Custom Claims Pattern
Firebase supports custom claims stored in the user's ID token. The admin claim is set server-side via Firebase Admin SDK and can be read client-side via `getIdTokenResult()`.

# Current Auth Pattern
- Zustand store in `src/store/authStore.js`
- Firebase auth initialized in `src/firebase.js`
- Auth modal shown in App.jsx when `!user`
